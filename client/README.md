# Client React — Jeux Quizz

Ce dossier contient une application React minimale (squelette).

Important:
- Aucun code WebSocket n'est fourni (conformément à la demande). Tu coderas côté serveur et client.

Commandes:
- npm install
- npm run dev

Structure:
- `src/main.jsx` — point d'entrée
- `src/App.jsx` — composant principal
- `src/pages/` — pages (Game, Admin)
- `src/data/questions.json` — questions de démonstration

Astuce: pour connecter ton serveur WebSocket, implémente la logique dans `src` lorsque tu seras prêt.
